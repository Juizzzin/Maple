# Maple
Snapshot and restore installed packages across systems using a portable compressed blueprint.

## Run the program
### Run the binary
Download the latest release for your system and run the binary

### Run with Python
```
git clone https://github.com/Juizzzin/Maple.git
cd Maple
python3 maple.py
```
